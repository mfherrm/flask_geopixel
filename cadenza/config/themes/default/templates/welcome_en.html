First steps in disy Cadenza
{% include './quick-intro.html' %}

1 Welcome Page

disy Cadenza's welcome page is divided into two parts. The Navigator on the left allows you to access and visualize the connected data. When launching Cadenza, the Navigator is always available. When opening a workbook or a single view, the Navigator automatically closes.

It can be reopened at any time by clicking the Navigator button in the main header.

2 Navigator

The Navigator provides a quick overview of the contents.

These are thematically grouped in different directories. They typically include:

  • Workbooks with corresponding worksheets
  • Directories with workbooks grouped by topic
  • Links

To help you find specific content, Disy Cadenza offers a search function. It is located in the upper part of the Navigator.

3 Demo Content

For demonstration purposes, this disy Cadenza installation contains data from the fields of Verkehr & Mobility (traffic and mobility), Gewaesserguete (water quality) and Personal-Controlling (personell controlling within directory "Zentrale Bereiche").

Examples of individual views as well as workbooks are provided in directory Gewaesserguete (water quality). English-language analyses are provided in directory "Verkehr & Mobilitaet (traffic & mobility)".

4 Workbooks

In disy Cadenza, workbooks are used for data visualization and systematic analysis. In this way, correlations can be identified and visualized in a comprehensible way.

A workbook consists of one or more worksheets containing the data prepared in a topic-based structure.

Click this symbol to create a new, empty workbook.

Ready-to-use workbooks are available in directories "Gewaesserguete (water quality)", "Verkehr & Mobilitaet (traffic & mobility)" and "Zentrale Dienste (central services)".

Click the arrow next to a workbook to list all available worksheets in the Navigator. By clicking a worksheet, it opens in disy Cadenza.

5 Worksheets

Various worksheets are listed in workbook "Uebersicht Messstellen (measurement stations overview)" .

The preceding symbols quickly indicate how the data is visualized on the worksheet. The following options are available:

  • Dashboard
  • Table
  • Chart
  • Map

6 Dashboard

A dashboard consists of several views that can simultaneously show different visualizations of the underlying data. In the "Total Registrations" worksheet of the "Registrations" workbook, for example, the data is presented in the form of two indicators, a column chart, a table with color highlighting and a map.

A text field provides additional information about the data. An image visually places data in context. Additional visualization types are available.

If you have permission to access the workbook, you can use the Data Manager to add additional data or change various data settings in the Designer, such as customizing the aggregation or display.

7 Filtering Data

The filters in the so-called Analysis Context on the left allow a precise data selection in the displayed views. Apply the filters in the Analysis Context of a workbook as needed to narrow down the data relevant to you. The worksheet views refresh automatically.

For more information on how to work with disy Cadenza and how to process data in a workbook, please refer to the Learning Modules.
