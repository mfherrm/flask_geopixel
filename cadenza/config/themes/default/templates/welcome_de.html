Erste Schritte in disy Cadenza

1 Startseite

Die Startseite von disy Cadenza ist zweigeteilt. Auf der linken Seite sehen Sie den sogenannten Navigator, über den Sie auf die angebundenen Daten zugreifen und diese visualisieren können. Auf der Startseite ist er immer geöffnet. Sobald Sie eine Arbeitsmappe oder eine Einzelsicht öffnen, wird der Navigator automatisch geschlossen.

Sie können den Navigator jederzeit wieder öffnen, indem Sie in der Hauptleiste auf die Schaltfläche „Navigator“ klicken.

2 Navigator

Der Navigator ermöglicht Ihnen einen schnellen Überblick über die zur Verfügung gestellten Inhalte.

Diese sind in unterschiedlichen Verzeichnissen thematisch gruppiert. Mögliche Inhalte sind:

  • Arbeitsmappen mit zugehörigen Arbeitsblättern
  • Verzeichnisse, um Arbeitsmappen thematisch zu bündeln
  • Links

Wenn Sie gezielt nach bestimmten Inhalten suchen möchten, bietet Ihnen disy Cadenza eine Suchfunktion. Diese befindet sich im oberen Bereich des Navigators.

3 Demo-Inhalte

In dieser disy Cadenza-Installation sind zu Demonstrationszwecken Daten aus den Themenbereichen Verkehr & Mobilität, Gewässergüte und Personal-Controlling (zu finden im Verzeichnis „Zentrale Bereiche") enthalten.

Im Verzeichnis Gewässergüte finden Sie Beispiele für Arbeitsmappen zu Gewässermessstellen in Niedersachsen. Englischsprachige Auswertungen werden im Verzeichnis „Verkehr & Mobilität / traffic & mobility“ bereitgestellt.

4 Arbeitsmappen

In disy Cadenza werden Arbeitsmappen verwendet, um einen Überblick über die Daten zu erhalten und diese gezielt zu analysieren. So können Zusammenhänge herausgearbeitet und verständlich visualisiert werden.

Eine Arbeitsmappe besteht aus einem oder mehreren Arbeitsblättern, in denen die Daten thematisch strukturiert aufbereitet sind.

Klicken Sie auf dieses Symbol, um eine neue, leere Arbeitsmappe zu erstellen.

Vorbereitete Beispiele für Arbeitsmappen finden Sie in den Verzeichnissen „Gewässergüte", „Verkehr & Mobilität / traffic & mobility" und "Zentrale Dienste".

Klicken Sie auf den Pfeil neben einer Arbeitsmappe, um alle darin enthaltenen Arbeitsblätter im Navigator anzuzeigen. Durch einen Klick auf ein Arbeitsblatt wird das betreffende Arbeitsblatt in disy Cadenza geöffnet.

5 Arbeitsblätter

In der Arbeitsmappe „Übersicht Messtellen" sehen Sie verschiedene Arbeitsblätter.

Anhand der vorangestellten Symbole können Sie auf einen Blick erkennen, wie die Daten auf dem Arbeitsblatt visualisiert werden. Es gibt folgende Optionen:

  • Dashboard
  • Tabelle
  • Diagramm
  • Karte

6 Dashboard

Ein Dashboard besteht aus mehreren Sichten, die gleichzeitig unterschiedliche Visualisierungen der zugrundeliegenden Daten zeigen können. Im Arbeitsblatt „Total Registrations" der Arbeitsmappe „Registrations" werden die Daten beispielsweise in zwei Indikatoren, einem Säulendiagramm, einer Tabelle mit farblicher Hervorhebung sowie in einer Karte aufbereitet dargestellt.

Ein Textfeld liefert weitere Informationen zu den Daten. Ein Bild ordnet die Daten visuell ein. Daneben stehen weitere Visualisierungstypen zur Verfügung.

Wenn Sie die entsprechenden Rechte an der Arbeitsmappe haben, können Sie mithilfe des Datenmanagers weitere Daten hinzufügen oder im Designer unterschiedliche Einstellungen an den Daten vornehmen, wie beispielsweise die Anpassung der Aggregation oder der Darstellung.

7 Daten filtern

Die Filter im sogenannten Analysekontext auf der linken Seite ermöglichen eine gezielte Datenauswahl in den dargestellten Sichten. Um die Daten der Arbeitsmappe auf einen für Sie relevanten Bereich einzuschränken, setzen Sie die entsprechenden Filter im Analysekontext. Die Sichten im Arbeitsblatt werden automatisch aktualisiert.

Ausführliche Informationen zu den Arbeitsmöglichkeiten in disy Cadenza und der Datenaufbereitung in einer Arbeitsmappe erhalten Sie in den Lernmodulen.
